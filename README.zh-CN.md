# auditor 智能体代码与合规审计（中文版）

系统化代码与合规审计框架（Phase G + S1-S5 阶段）。包含执行协议、质量属性和自我学习能力。

🌐 **语言**: [English](README.md) | **简体中文**（当前）

## 描述

为 OpenClaw 框架提供系统化审计能力。

### 核心特性

- **Phase G + S1-S5 流程**：从准备到总结的完整审计流程
- **执行协议**：标准化的审计步骤
- **质量属性评估**：可量化的审计维度
- **自我学习**：从历史审计中提炼经验
- **多语言支持**：审计报告可中英双语输出
- **合规检查**：内置常见合规项

### 适用场景

- 代码审计与代码评审
- 合规审计（财务、法律、安全）
- 知识库质量审计
- 流程审计与改进

## 安装

将整个 auditor/ 目录复制到 OpenClaw 技能目录（通常是 ~/.openclaw-skills/auditor/）。

## 目录结构

\auditor/
├── SKILL.md          # 技能主定义（先读这个）
├── references/       # 参考文档
├── scripts/          # 可执行工具
├── _knowledge/       # 知识库
└── _meta.json        # 技能元数据
\
